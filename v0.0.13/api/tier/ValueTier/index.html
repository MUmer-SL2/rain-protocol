ValueTier

Details#

A contract that is ValueTier expects to derive tiers from explicit values. For example an address must send or hold an amount of something to reach a given tier. Anything with predefined values that map to tiers can be a ValueTier.

Note that ValueTier does NOT implement ITier. ValueTier does include state however, to track the tierValues so is not a library.
